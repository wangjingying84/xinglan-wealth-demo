一条共同法理主线（十一案）

  • 案一 星澜：控制权人格化 + 个人担保打通家企 → 控制权机构化（M1/M3）+ 隔离做实（R3–R9）。
  • 案二 宏图：产权系于代持/隐名、控制权随继承裂变 → 代持还原 + 章程继承开关（公司法90）+ 信托存续替换继承事件。
  • 案三 瑞宁：表决权“委托给个人行使”一死即终止（民934）→ 改为“归入信托财产管理”（信托法52），控制不中断。
  • 案四 景行：离婚析产击穿的是“装进信托之前”的权属 → 风险在设立环节；敏感期忌单方入信托（解释二7/3条）。
  • 案五 安邦：人在境内即全球征税，离岸≠离税≠不申报；保留控制越多→隔离越弱 且 CRS穿透申报越确定；港股 WVR 可正面分离控制/收益。
  • 案六 宸光：身故有继承兜底，失能是“活体真空”——意定监护 vs 法定监护、信托不因失能终止（信托法52）、人钱分离三分防滥权。
  • 案七 嘉沃：方向相反——保护信托对抗“受益人自己”。受益权默认可被执行（信托法47），靠全裁量 + 挥霍信托条款防御，效力无判例。
  • 案八 泰昇：对手是公权力。没收/追缴≠债权人撤销，是特定标的追及；以非法财产设立信托自始无效（信托法11）。
  • 案九 衡远：纠纷根源是多文件冲突、或关键一份从未生效——生前信托主轴 + 遗嘱兜底排除 + 保单独立轨 + 遗产管理人（五步SOP）。
  • 案十 仁济：境内“做慈善+保留控制/收益”空间极窄——基金会控股被慈善法55条封堵；慈善信托公益绑定，双轨分离、影响力止于治理。
  • 案十一 安和：受益人是终身需照护的残障/未成年人——特殊需要信托 + 遗嘱指定监护接续 + 多方制衡防侵占 + 给付节奏化。

境内“既做慈善又保留控制/收益”空间极窄：基金会控股反哺家族被慈善法55条封堵；慈善信托可不设具体受益人但被公益绑定（家族不能受益）。正解＝慈善信托（公益）+ 家族信托（私益）双轨，影响力止于受托人/监察人席位、捐赠人建议权、命名权——不及私益处分，并用足税收优惠。
